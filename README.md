# Linux configs 

for bash, screen, vim etc

## Usage

### Bash
Add this to your **~/.bashrc**
```bash
theme='main'
disableTouchpad=1
source ~/path/to/cloned/repo/bash/.bashrc
```

### vim
```bash
source ~/path/to/cloned/repo/vim/.vimrc
```

### screen
```bash
source ~/path/to/cloned/repo/screen/.screenrc
```
